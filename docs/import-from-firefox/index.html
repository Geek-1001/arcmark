Import Firefox Bookmarks

Step 1: Export from Firefox

  1. Open Firefox
  2. Click the menu button (three horizontal lines) in the top-right
  3. Click Bookmarks, then Manage bookmarks at the bottom
  4. In the Library window, click Import and Backup
  5. Select Export Bookmarks to HTML...
  6. Choose a location and click Save

Step 2: Import to Arcmark

  1. Open Arcmark
  2. Go to Settings (gear icon)
  3. Scroll to the Import & Export section
  4. Click Import from Chrome, Safari, Firefox
  5. Select the HTML file you exported from Firefox
Note: Your folder structure will be preserved during import. A new workspace called "Imported Bookmarks" will be created with all your imported bookmarks and folders.
