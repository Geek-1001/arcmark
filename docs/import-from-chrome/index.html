Import Chrome Bookmarks

Step 1: Export from Chrome

  1. Open Google Chrome
  2. Click the three-dot menu in the top-right corner
  3. Go to Bookmarks and lists
  4. Click Bookmark manager
  5. In the Bookmark manager, click the three-dot menu in the top-right
  6. Select Export bookmarks
  7. Choose a location and save the HTML file

Step 2: Import to Arcmark

  1. Open Arcmark
  2. Go to Settings (gear icon)
  3. Scroll to the Import & Export section
  4. Click Import from Chrome
  5. Select the HTML file you exported from Chrome
Note: Your folder structure will be preserved during import. A new workspace called "Chrome Bookmarks" will be created with all your imported bookmarks and folders.
